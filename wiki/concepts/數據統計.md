# 數據統計與圖表製作 (Data Statistics & Visualization)

科學數據必須經過統計檢定，才能證明其差異並非隨機誤差。利用 Julius.AI 與 Claude Artifacts，學生能獨立完成專業的推論統計與互動圖表。

---

## 📊 數據統計的核心步驟

### 1. 敘述統計 (Descriptive Statistics)
- **計算指標**：平均值（Mean）、中位數（Median）、標準差（Standard Deviation, SD）、標準誤差（Standard Error of the Mean, SEM）及樣本數（n）。
- **目的**：了解數據的基本分布與離散程度。

### 2. 推論統計 (Inferential Statistics)
- **配對 t 檢定 (Paired t-test)**：比較同一組樣本在處理前後（如運動前後的心拍數）是否有顯著差異。
- **獨立 t 檢定 (Independent t-test)**：比較兩組獨立樣本（如實驗組與對照組）的平均值差異。
- **單因子變異數分析 (One-way ANOVA)**：比較三組或以上獨立樣本（如三種溫度下的萌發率）的平均值是否有顯著差異。若顯著，需進一步進行事後比較（Post-hoc test，如 Tukey HSD）。
- **推論核心**：所有統計結果應報告 $p$ 值（$p < 0.05$ 為顯著差異）、$F$ 值或 $t$ 值，以及效應量（Effect Size）。

## 🛠️ AI 輔助統計工具鏈

### 1. Julius.AI：專業生物統計
- **使用方法**：上傳 Excel/CSV 檔案，以對話方式指示 AI 執行特定統計檢定。
- **優勢**：
  - 自動生成 Python 程式碼在後台執行運算，確保計算零幻覺。
  - 能輸出標準的 ANOVA 表格、產出符合期刊投稿規範的箱型圖（Box Plot）、散布圖（Scatter Plot）或熱力圖（Heatmap）。
  - 提供專業的統計學原理解釋。

### 2. Claude Artifacts：互動式網頁圖表
- **使用方法**：提供數據，請 Claude 使用 React + Recharts 或 D3.js 寫成單一網頁元件。
- **優勢**：
  - 可生成具備下拉選單、滑鼠懸停顯示變化率、異常值紅色警示等互動功能的圖表。
  - 非常適合在網頁版科展簡報、口頭發表或線上展示時演示。

---

## ⚠️ 數據誠實原則
- **異常值處理**：數據中若有極端值，應有明確的學術理由（如實驗失誤、儀器故障）才能排除，且必須在報告中說明「排除/保留異常值」的前後差異，絕不可隨意刪除數據以迎合預期結論。
- **圖表誠實**：Y 軸起點通常應從 0 開始，避免刻意誇大微小差異；必須加上誤差線（Error Bars，如標示 SD 或 SEM）。
