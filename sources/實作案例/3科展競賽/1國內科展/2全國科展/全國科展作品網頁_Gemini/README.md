# 小孢子靈芝蛋白質（GMI）協同臨床用藥 (Dexamethasone) 可有效抑制脂多醣誘導的發炎作用 - 程式視覺化平台

本專案將中華民國第 66 屆中小學科學展覽會（動物與醫學學科）作品說明書的內容，製作為高互動性且極具視覺質感的程式視覺化網頁。

專案旨在輔助評審快速理解研究的分子機制、實驗方法及數學建模，並透過動態數據探索與協同效應模擬器，以直觀且科學的方式展現研究成果。

## 核心功能

1. **「雙重路徑阻斷」細胞通路 SVG 互動動畫**
   - 視覺化展示 LPS 刺激巨噬細胞引發的發炎訊號鏈，以及 GMI（上游細胞質）與 Dexamethasone（下游細胞核）的包夾阻斷機制。
   - 與藥物配方模擬器即時連動，動態調整虛線傳導流速與發炎因子釋放量。

2. **藥物配方 Bliss 模型模擬器**
   - 提供 GMI 與 DEX 濃度的雙向滑桿控制。
   - 即時套用 Bliss 獨立模型數學公式 $E_{expected} = E_D + E_G - E_D \times E_G$，並與實驗真實抑制率進行對比。
   - 自動判斷藥物作用關係，即時點亮 **Synergy (協同作用)**、**Additive (加成作用)** 或 **Antagonist (拮抗作用)** 標籤，並針對高劑量類固醇提供細胞壓力預警。

3. **實驗數據多維度探索器 (Chart.js)**
   - 互動式圖表展示：
     - **MTT 細胞存活率**（可切換 GMI 單獨、DEX 單獨與聯合用藥影響）。
     - **IL-6 分泌量定量**（以直條圖對比 CTL、LPS、DEX單用、GMI單用與聯合處理在低/中/高劑量下的分泌數值）。
     - **Bliss 協同趨勢折線圖**（展示不同 GMI 濃度下，DEX 濃度對比理論極限的協同平台曲線）。

## 如何在本地運行

因為本專案使用了 CDN 來載入 **Chart.js** 與 **FontAwesome** 圖示庫，建議在有網路連線的環境下開啟網頁。

### 方式 1：直接開啟 (推薦快速檢視)
- 在雙擊或在瀏覽器中直接開啟本機的 `index.html` 檔案即可運行。

### 方式 2：使用本地 Web 伺服器 (最佳相容性)
如果您已安裝 Node.js 或 Python，可以在專案根目錄下使用終端機開啟輕量化 Web 伺服器：

**使用 Python 3：**
```bash
python3 -m http.server 8000
```
然後在瀏覽器中開啟 [http://localhost:8000](http://localhost:8000)。

**使用 Node.js (`npm`):**
```bash
npx serve
```
